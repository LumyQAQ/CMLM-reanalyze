CMLM (城门立木) 是一套全自动化的 A 股量价扫描与可视化看板系统。通过对全市场 (涵盖主板 10CM、创业板/科创板 20CM) 5000+ 标的进行极速穿透，精准捕捉主升浪启动节点与极致缩量洗盘节点。

## 💡 核心交易逻辑：双模驱动

本系统专为超短/波段交易者设计，包含两大漏斗模型：

### 🚀 模块一：右侧主升 ——【龙抬头】(强势放量突破)
* **适用场景：** 寻找当日资金合力做多、量价齐升的异动标的。
* **算法引擎：** 严格筛选今日涨幅在 2.5%~5% 以上，且成交量较过去 15 日平均常态均量放大 **1.25 倍至 1.35 倍**以上的个股。
* **逻辑标签：** 内置分类算法，智能打标为 `强势放量突破`、`趋势放量异动`、`回调二波起涨`。

### 🐉 模块二：左侧低吸 ——【龙回头】(极致缩量洗盘)
* **适用场景：** 寻找前期爆发过，目前正在分歧洗盘，寻找高胜率买点的潜伏标的。
* **算法引擎：** 1. 寻找近 2~6 天内出现过涨幅 >5% 且放量 >1.5倍 的“爆发阳线”。
  2. 寻找近两日出现 **极致缩量** (今日量 < 昨天量，且小于爆发日 65%)。
  3. 寻找未破防线 (今日收盘下跌洗盘，但绝不跌破爆发日开盘价)。
* **防守底线：** 剔除单日跌幅超过 -7% 的破位标的。

## 🏗️ 架构亮点：前后端极速解耦

针对云端服务器跨境拉取国内 A 股节点容易触发“断流、假连通”的痛点，V4.4 版本采用了最先进的**【前后端物理隔离】**架构：

1. **💻 数据引擎层 (本地化 `v4_engine.py`)：**
   * 部署在本地高配机器，享受国内极速网络。
   * 调用 `mootdx` 智能测速寻找最快节点，**1 分钟内完成全市场 3000+ 有效标的扫板与切片计算**。
   * **【独家】盘中动态量能外推雷达：** 引擎内置系统时间感知。若在 14:30 盘中运行，系统会自动按 `240/已开盘分钟数` 动态放大当前成交额，完美支持**尾盘抢先知**与盘中缩量预估，不再局限于盘后复盘。
2. **☁️ 展示看板层 (云端 `v4_surge_board.py`)：**
   * 托管于 Streamlit Cloud，纯前端展示，**零延迟秒开**。
   * 搭载超强“容错装甲”：完美处理空数据壳（行情极差时无标的入选不崩溃）、自动修正海外服务器的 UTC 时区偏移（精准显示东八区更新时间）。
3. **🌉 数据桥梁 (GitHub)：**
   * 本地计算出 `.csv` 数据后，通过 Git 极速推送到云端，驱动看板刷新。
